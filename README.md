# A quick build web app with ReactJS, Node, and Express.
To run the app please clone the repo, run the Node server and the front-end with the following command(run in the main repo):
```console
foo@bar:~$ npm install
foo@bar:~$ cd server
foo@bar:~$ npm start
foo@bar:~$ cd ../app
foo@bar:~$ npm install
foo@bar:~$ npm start
```
